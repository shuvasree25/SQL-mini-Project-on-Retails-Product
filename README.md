# SQL-mini-Project-on-Retails-Product
A beginner-friendly SQL project demonstrating fundamental SQL concepts by analyzing a retail products dataset using PostgreSQL.
📌 About the Project                                                                                                                                                This project analyzes a custom retail products dataset that simulates a small retail store's product catalog with 50 products across Electronics, Clothing, Grocery, Home & Kitchen, and Beauty. The dataset was created from scratch and includes 42 SQL queries covering fundamental SQL concepts and business-oriented analysis using PostgreSQL.
🛠️ Concepts Covered                                                                                                                                                 SELECT – Retrieving specific columns and using column aliases
WHERE – Filtering data using =, >, IN, BETWEEN, and LIKE                                                                                                   Aggregate Functions – COUNT(), SUM(), AVG(), MAX(), and MIN()                                                                                                     GROUP BY – Summarizing data by category and brand                                                                                                                 HAVING – Filtering grouped and aggregated result                                                                                                               ORDER BY – Sorting query results in ascending or descending order                                                                                                LIMIT – Retrieving a specified number of records
